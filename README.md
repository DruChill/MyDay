## 🚀 Acerca del Proyecto

**MyDay** es una aplicación Android de código abierto diseñada para simplificar la gestión de tareas diarias. Construida con tecnologías modernas, MyDay aprovecha la potencia de la nube para mantener tus datos sincronizados en tiempo real.

El objetivo es crear una experiencia de usuario fluida, minimalista y libre de distracciones.



[Image of mobile app mockup generic]


### ✨ Características Principales

* 🔐 **Autenticación Segura:** Inicio de sesión con correo y Google (Firebase Auth).
* ☁️ **Sincronización en Tiempo Real:** Tus tareas siempre actualizadas (Cloud Firestore).
* 🎨 **Diseño Moderno:** Interfaz limpia siguiendo Material Design 3.
* 🌙 **Modo Oscuro:** Totalmente compatible con el tema del sistema.
* 🔔 **Recordatorios:** Notificaciones locales para que no olvides nada.
